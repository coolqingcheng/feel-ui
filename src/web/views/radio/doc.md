### 禁用单选框
使用disables属性传递一个数组，禁用单选框组中的部分选择项，使部分选择项无法选中

<demo path="./RadioDemo2.vue"></demo>

# 属性

| 名称     | 说明                                          | 类型  | 可选值 | 默认值 |
| -------- | --------------------------------------------- | ----- | ------ | ------ |
| data     | 选择项列表,传递一个数组，如:['支付宝','微信'] | array |        |        |
| disables | 禁用项目，禁用一个或者多个选择项              | array |        |        |

# 事件

| 名称   | 说明                             | 返回值         |
| ------ | -------------------------------- | -------------- |
| select | 选中某个项的时候，会触发当前事件 | {value:返回值} |
