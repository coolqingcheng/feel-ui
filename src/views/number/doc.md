# 属性

| 名称    | 说明                     | 类型   | 默认值 |
| ------- | ------------------------ | ------ | ------ |
| step    | 步长，每次增加或减少的值 | number |        |
| v-model | 双向绑定值               | number |        |
| min     | 最小值                   | number |        |
| max     | 最大值                   | number |        |



# 事件

| 名称   | 说明           | 返回值    |
| ------ | -------------- | --------- |
| change | 值修改时候触发 | {value:1} |
